# Listas de listas y Diccionarios

## Listas de listas (Matriz)

También se conoce como:

- Lista de listas
- Matrices
- Estructura bidimensional
- Tabla de filas y columnas

Ejemplo conceptual:

| Producto | Precio | Cantidad |
|----------|--------|----------|
| Arroz    | 1200   | 5        |
| Leche    | 900    | 10       |
| Pan      | 700    | 8        |

Representación en Python:

```python
inventario = [
    ["Arroz", 1200, 5],
    ["Leche", 900, 10],
    ["Pan", 700, 8]
]
```

* **Sintaxis:** Se abren corchetes generales, y adentro se separan por comas otros corchetes más pequeños.
```python
estante_principal = [
    ["Manzanas", "Peras", "Uvas"],      # Esta es la Fila 0
    ["Leche", "Queso", "Mantequilla"],  # Esta es la Fila 1
    ["Jabón", "Cloro", "Esponjas"]      # Esta es la Fila 2
]
```

### 2. ¿Cómo acceder a los datos? (Doble Índice)

Ambos índices empiezan a contar desde cero. `estante_principal[fila][columna]`.

```python
    print("--- SISTEMA DE BÚSQUEDA POR COORDENADAS ---")
    
    bodega = [
        ["Arroz", "Frijoles", "Lentejas"], # Fila 0: Granos
        ["Atún", "Sardina", "Sopa"],       # Fila 1: Enlatados
        ["Agua", "Jugo", "Refresco"]       # Fila 2: Bebidas
    ]
    
    # Queremos sacar la "Sardina".
    # Paso 1: ¿En qué fila está? En la fila 1 (Enlatados).
    # Paso 2: ¿En qué columna está dentro de esa fila? En la columna 1.
    producto_buscado = bodega[1][1]
    print("Producto extraído:", producto_buscado)
    
    # Queremos sacar el "Agua". (Fila 2, Columna 0)
    print("Bebida seleccionada:", bodega[2][0])
    
    # Modificar un dato: El Jugo se agotó, pondremos Té.
    bodega[2][1] = "Té"
    print("Fila de bebidas actualizada:", bodega[2])
```

# Recorrer una matriz
Necesitamos un **bucle anidado** (un `for` dentro de otro `for`). El primer bucle se encarga de bajar por los pisos (filas), y el segundo bucle se encarga de caminar de izquierda a derecha revisando los productos de ese piso (columnas).


La estructura siempre requiere dos variables temporales. Una para la lista grande (la fila) y otra para la lista pequeña (el elemento individual). Recuerda respetar la doble sangría para el segundo bucle.
```python
    for fila in matriz_principal:
        for elemento in fila:
            # Acción a realizar con el elemento
```
```
inventario[fila][columna]

- Primer índice → Fila
- Segundo índice → Columna
```
